# FormGrid

## How to use

```
import { FormGrid } from '@ederzadravec@jade-ui'

...

<FormGrid config={[]} />
```

## Props

```
  config = [                        // Rows
    [                             // Columns
      {                           // Column
        schema: 'field1',         // name, id, key
        size: { md: 6 },          // responsible width (xs, sm , md, lg, xl),
        hide: () => {},           // return true to hide,
        type: TextInput,          // Component to render
        props: (schema) => ({})   // props of component
      }
    ]
  ]
```
