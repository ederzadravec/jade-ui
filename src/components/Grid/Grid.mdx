# Grid

## How to use

```
import { Grid } from '@ederzadravec@jade-ui'

...

<Grid container>                                // Row/Container
  <Grid size={{ md: 12 }}> CONTENT </Grid>      // Column
</Grid>
```

## Props

```
container   bool      false         // Row/Container of columns
size        {}        { md: 12 }    // Size of column

```
